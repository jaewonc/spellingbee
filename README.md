# Spelling Bee Practice

A web-based spelling practice application with audio pronunciation and word lists.

## Features

- Audio pronunciation of words
- Progress tracking
- Word list management from CSV files
- Responsive design with background blur effects

## Usage

Open `spelling-bee.html` in a web browser.

## File Structure

- `spelling-bee.html` - Main application
- `background.jpg` - Background image
- `wordlists/` - CSV word lists by year
